Модуль: Аналіз електронної пошти / Аналіз інцидентів
Сценарій: Fake Invoice (Emotet-style XLSM Dropper)
Формат: Self-Paced
Версія документу: 1.0

---

## 1. Що вам знадобиться

* Комп'ютер з доступом до мережі лабораторії
* SSH-клієнт (вбудований у Linux/macOS або PuTTY для Windows)
* Браузер (Chrome, Firefox або Edge)

---

## 2. Підключення до веб-поштового клієнта

Відкрийте браузер та перейдіть за адресою:

```
http://192.168.88.202:8086
```

Тут знаходиться фішинговий лист для аналізу. Додаткова авторизація не потрібна.

---

## 3. Підключення по SSH

### Linux / macOS

Відкрийте термінал та виконайте:

```bash
ssh student1@192.168.88.202
```

Введіть пароль: `student1`

### Windows (PuTTY)

1. Відкрийте PuTTY
2. У полі **Host Name** введіть: `192.168.88.202`
3. У полі **Port** введіть: `22`
4. Натисніть **Open**
5. Введіть ім'я користувача: `student1`
6. Введіть пароль: `student1`

### Windows (PowerShell / CMD)

```
ssh student1@192.168.88.202
```

## 5. Після підключення по SSH

Після входу ви автоматично опинитесь у домашній директорії. Перевірте доступ до лабораторних матеріалів:

```bash
# Переглянути файли лабораторії
ls ~/scenario/

# Проаналізувати трафік
tcpdump -r ~/scenario/sc06.pcap host 185.156.72.11

# Зайти до контейнера
docker exec -it forensics_sc06_invoice bash
```

## 6. Швидкий старт — що робити першим

| Крок | Дія | Де |
|---|---|---|
| 1 | Відкрити веб-поштовий клієнт | Браузер → http://192.168.88.202:8086 |
| 2 | Підключитись по SSH | Термінал → ssh studentN@192.168.88.202 |
| 3 | Аналізувати трафік | SSH сесія → tcpdump -r ~/scenario/sc06.pcap |
| 4 | Переглянути журнали | SSH сесія → docker exec → cat /var/log/... |

## 7. Якщо щось не працює

**Веб-інтерфейс не відкривається:**
```bash
# Перевірити чи запущені контейнери (від analyst)
docker compose -f ~/scenario/docker-compose.sc06.yml ps
```

**emlAnalyzer не знайдено:**
```bash
# Переконатись що встановлено системно
emlAnalyzer -i ~/scenario/emails/06_malware_invoice.eml
```

**Немає доступу до файлів сценарію:**
```bash
ls ~/scenario/
# Якщо порожньо — звернутись до викладача
```
