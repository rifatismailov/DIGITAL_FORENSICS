## 3. Передісторія атаки

### 3.1 Розвідка зловмисника (за 2–3 тижні до атаки)

* Зловмисники знайшли профіль Олени Коваль у LinkedIn — посаду «бухгалтер TechFrontier UA»
* Через корпоративний сайт визначили формат поштових адрес компанії
* Через Prozorro знайшли тендерні документи з назвами контрагентів
* Зареєстрували домен ukr-accounting-service.net за 18 днів до атаки (2024-03-09)
* Налаштували SMTP-сервер на IP 193.201.22.41 (Румунія, захищений хостинг для зловмисників)

### 3.2 Підготовка фішингового листа

* Підроблений відправник: «Бухгалтерія УкрОблік Сервіс» — справжня бухгалтерська фірма
* Тема (закодована у base64): «TERMINOVO: Рахунок №INV-2024-0847 до оплати» — штучна терміновість
* Фіктивний рахунок-фактура на 74 850 грн — реалістична сума для бухгалтера
* Вкладення: Рахунок_фактура_INV-2024-0847.xlsm — Excel із прихованим XLM-макросом
* Кнопка «Переглянути онлайн» з посиланням на C2-домен замість легітимного Google Docs
* Піксель відстеження розміром 1×1px для підтвердження відкриття листа

## 4. Ситуація на момент початку лабораторної роботи

О **09:59** система Defender for Endpoint надіслала критичне сповіщення:

```
[КРИТИЧНО] INC-2024-032701
Хост:        DESKTOP-ACCOUNTING01 (192.168.5.88)
Користувач:  o.koval@techfrontier.com.ua
Сповіщення:  Backdoor.Emotet.Gen.Dropper — виявлено C2-маяки
C2:          185.156.72.11:8080 (8 з'єднань за 18 хвилин)
```

Виявлені підозрілі події:

* EXCEL.EXE → cmd.exe → powershell.exe (аномальне дерево процесів)
* Завантаження файлів .ps1 та .dll із зовнішнього C2-домену
* Запис ключа Run у реєстрі для закріплення в системі
* Спроба ексфільтрації банківських даних (заблоковано DLP)
* Сканування внутрішньої мережі (DC01, DC02, ERP01)

Хост продовжує надсилати маякові запити кожні ~120 секунд. Олена Коваль не підозрює про компрометацію.
